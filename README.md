# Currency Calculator

## JavaFX application, parses JSON objects from <a href src="http://api.nbp.pl/">NBP Web API</a> to get currency exchange rates. Calculates specific amounts of a currency to buy or sell.

### Built with:
* Java
* JavaFX
* Spring Core
* Jackson
* Maven

### Tested with:
* Junit
* Mockito
